# Plasma-Bank
Install git
Execute the following command:
  git clone --depth 1 https://github.com/Shahjalal-Sakil/Plasma-Bank.git
  npm install mongodb
  npm install mongoose
  mongod
  npm intsall --save-dev nodemon
  npm install express
  npm run start
 check the api using postman:
    http://localhost:8080
  
  for donor:
    /donors
   for patient:
   /patients
